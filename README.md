# Android Course - Jetpack Compose Learnings

Welcome to the **Android_Course** repository! This project serves as a structured workspace for learning and practicing modern Android application development using **Kotlin** and **Jetpack Compose**.

    * `TextExample`: Styling Compose `Text` with colors, fonts, and letter spacing.
    * `TextFieldExample`: Using `TextField` with leading/trailing icons, custom shapes, and remember-based state tracking.
    * `OutlinedTextFieldExample`: Creating an input field with an outlined border and custom rounded corners.

## đźš€ Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/RibhuMitra/Android_Learnings.git
   ```
2. **Open in Android Studio**:
   * Choose `Open` -> Select the `Android_Course` folder.
   * Allow Gradle build to sync dependencies.
3. **Run the App**:
   * Set up an Android Emulator or connect a physical device via USB.
   * Click **Run** (`Shift + F10` or the green Play icon).
